- **Web:** โจมตีช่องโหว่ประเภท IDOR และ Path Traversal ในระบบจัดการไฟล์
- **Pwn:** การทำ Buffer Overflow พื้นฐานและการทำ Ret2Libc บนระบบ 64-bit
- **Forensics:** วิเคราะห์หน่วยความจำ (Memory Forensics) และการกู้คืนไฟล์จาก Disk Image
- **Crypto:** ถอดรหัสข้อความที่ใช้ XOR แบบซ้ำๆ และการวิเคราะห์ RSA Public Key
